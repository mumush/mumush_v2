The Sudo User

Sudo (Super User Do) is a program that is meant to allow certain users to execute certain commands as root. The purpose of these commands is to allow people to have as few privileges as possible while still allowing users to get their work done. Each sudo user has a username and password and their account is granted a certain number of privileges.This limitation not only limits the damage that mistakes can make, it also people to view logs of who ran what command and when.

The Root User

A root user is the equivilent of the administrator user on Windows. It can use commands to modify the system in any way desired and to grant and take away permission. However, this also means that you have to be very careful with what you do since with unrestricted access, you could potential do permanent damage to your system.

Basic Commands

ls

Lists all of the files in the current directory

emacs filename

Opens an editor that lets you create and edit a file

mv filename1 filename2

Moves a file from filename1 to filename2

cp filename1 filename2

Copies the file contents from filename1 to filename2

rm filename

Removes filename

chmod options filename

Allows you change the read, write, and execute permissions on your files

mkdir directoryname

Makes a new directory in the current working directory

cd dirname

Moves you to the dirname directory

pwd

Returns the name of the current directory being worked in
